# 快速开始

# 编写一个简单的命令脚本

首先,找到userData文件夹中的command.js并打开它
随意定义一个变量并编写命令,这里使用了对象解构语法,接收一个object,并将它的t键的value提取出来,赋值给text
myCommend是为了更好的了解命令的功能而定义的函数名,但是要注意的是,由于需要使用this访问引擎的内部变量,不能使用匿名(箭头)函数

let mc = function myCommend({t:text}) {
  console.log(text);
}

找到将commands变量,将命令注册到引擎中
这里使用了对象的简写形式,它相当于{ mc: mc }

const commands = { mc }

现在,你已经可以在剧本中使用这个命令了

# 在剧本中使用命令

编写剧本的方法在这里不再赘述,现在在你的剧本中添加以下内容

@mc t:"Hello World!"

在引擎运行到命令所在的幕时,控制台将输出Hello World!

# 通过定义宏使命令更可读

如果你不了解宏的使用,你可能需要先去阅读基础 · 编写剧本中的别名(宏)部分 现在,向macro.txt中添加以下内容

@输出 文字:{text} => @mc t:{text}

这样你就可以使用更可读的形式编写剧本了

@输出 文字:"Hello World!"

# 运行原理

可以发现,mc对应你注册在引擎中的命令名称,t对应{t:text}中的t
引擎实际上会将这行脚本解析为这样一个对象{ "@": "mc", "t": "Hello World!" },并传入命令函数中
而如果进行了宏的配置,在进行脚本解析时会先进行宏的替换,{ "@": "输出", "文字": "Hello World!" }被处理为{ "@": "mc", "t": "Hello World!" },之后的流程照常进行
