# 快速开始

# 需要了解的概念

# 命令:

参考接下来的"命令"部分,操作某个元素(如音乐,图片等)的指令就称为命令

# 幕:

一幕的命令执行完毕后，将等待用户点击后再执行下一幕

# 剧本基本格式

星夜引擎的剧本基本格式如下所示

----
@命令名称 命令参数=参数值 命令参数=参数值 ///注释
----
@命令名称 命令参数=参数值 命令参数=参数值
----

----用于分割不同的幕
///用于注释,在一行之内,///之后的内容会被忽略

举个实际栗子

----
@背景 文件=/saltlife.jpg/
@立绘 名称=/咸鱼/ 文件:/fish.jpg/ x:640 y:640
----
@对话 名称=/咸鱼/ 文本=/啊这...\n摸了!/ ///完工,睡觉
----

命令参数的顺序是随意的,一行也可以有多条命令,所以下面这样写也是正确的
不过,推荐每行只写一条命令,并且使用一个固定的参数顺序

@背景 文件=/saltlife.jpg/ @立绘 名称=/咸鱼/ 文件:/fish.jpg/ x:640 y:640

# 参数类型

星夜引擎的命令参数有三种类型,分别为文本 数字 条件

# 文本

一段文本,需要使用//或""包裹
在文本中建议使用\n来换行,也支持直接换行
在文本中,\写为\\
如果想在""包裹的文本中使用",需要写成\"
如果想在//包裹的文本中使用/,需要写成\/
这在编程世界被称为转义字符,是一种常见的避免歧义的方式
在这里提到的转义字符应该能满足你的需要,可以阅读转义字符章节获取更多信息

# 数字

一个数字,如50、1.2

# 条件

true(是)或false(否)

# 常见参数

星夜引擎有一些常见的参数命名,以下说明它们代表的含义

# 名称

给当前命令操作的元素命名
这个参数是文本类型

# 目标

在为元素命名后,使用这个参数来明确"我要操作目标元素"
这个参数是文本类型

# 文件

告诉命令要使用哪个文件,可以使用/表达引擎目录下的子文件夹
比如saltfish/fish.jpg代表引擎目录下saltfish文件夹中的fish.jpg
这个参数是文本类型

# 时间

持续时间,淡入时间,等待时间等等,都属于这个参数
目前,这个参数的单位是毫秒,1 秒等于 1000 毫秒
这个参数是数字类型

# 缓动

缓动函数是一种算法,可以实现淡入淡出等效果
本引擎使用anime.js实现缓动效果
可以参考缓动函数速查表 (opens new window),填入链接展示的缓动函数之一,如easeInSine
或者,也可以查看animejs 参考文档 (opens new window)中的Easings篇,了解本引擎支持的全部缓动种类
这个参数是文本类型

# 设置背景

# 命令名称@背景

参数 功能
文件 要作为背景的图片文件
w 初始宽度,默认值为图片自身宽度
h 初始高度,默认值为图片自身高度
x 初始横坐标,默认值为 0(画面左上角)
y 初始纵坐标,默认值为 0(画面左上角)
sx 初始横轴方向的缩放比例，默认值为 1(不缩放)
sy 初始纵轴方向的缩放比例，默认值为 1(不缩放)
r 初始图片旋转角度，默认值为 0(不旋转)
op 初始透明度,默认值为 1

其中只有文件参数是必须的

# 使用演示

@背景 文件:"咸鱼池塘.jpg" 缓动:"easeInQuad" 时间:500 x:-120

# 放置立绘

# 命令名称@立绘

参数 功能
名称 角色名称
文件 要使用的图片文件
w 初始宽度,默认值为图片自身宽度
h 初始高度,默认值为图片自身高度
x 初始横坐标,默认值为 0(画面左上角)
y 初始纵坐标,默认值为 0(画面左上角)
z 初始图层高度,默认值为 1(背景的图层高度 0)
sx 初始横轴方向的缩放比例，默认值为 1(不缩放)
sy 初始纵轴方向的缩放比例，默认值为 1(不缩放)
r 初始图片旋转角度，默认值为 0(不旋转)
op 初始透明度,默认值为 1

其中只有名称和文件参数是必须的

# 图像变换

# 命令名称@变换

参数 功能
目标 要操作的立绘的角色名称
缓动 变换效果要使用的缓动函数(默认值为 linear 线性)
时间 变换效果的持续时间(默认值为 0)

变换命令可以使用以下参数,如果你需要更强大的功能,也可以参考基本命令章节

参数 功能
w 宽度
h 高度
x 横坐标
y 纵坐标
sx 横轴方向缩放比例
sy 纵轴方向缩放比例
r 图片旋转角度

变换效果默认使用相对坐标,比如,立绘本身具有 x:100,再使用变换命令设置 x:100,最后的结果则会是 x=200 如果对同一个目标应用多个变换命令,后应用的变换命令将在之前的变换执行完毕后执行

# 使用演示

# 播放音频

# 命令名称@音频

参数 功能
类型 要操作的音轨的类型
文件 要使用的音频文件
名称 要操作的音轨的名称
循环 音频是否循环播放,默认值为 false(不循环)
时间 音频缓入缓出效果的持续时间

其中名称、循环、时间可选

如果省略名称,则它的默认名称和类型相同

如果已有相同名称的音频存在,会覆盖原先的音频

设置时间参数会为新音频应用缓入效果,为旧音频应用缓出效果

类型可以是BGM(背景音乐) SE(音效) Clip(角色语音) UISE(按钮音效)

设置为某个类型的含义是将该音频轨道和对应类型的音量、中断等设置绑定

# 使用演示

@音频 类型:"BGM" 文件:"bgm01.mp3" 循环:true 时间:1000

# 使用智能提示

只需要安装VSCode,并在 VSCode 中安装插件,即可使用提示功能
在剧本中输入@即可获得所有命令的提示,用 ↑↓ 键选择一个命令,按 Enter 键输入
这时光标会自动移动到第一个参数处并给出可能的提示,输入参数的值,按 Tab 键即可切换到下一个参数
这个插件还提供一定的语法高亮和错误检查功能

另外,VSCode 本身也支持一些方便的快捷键
你或许会用到以下快捷键

向上/向下移动行 Alt+ ↑ / ↓
向上/向下复制行 Shift+Alt + ↓ / ↑
在文件内查找替换 Ctrl + F

# 命令流

# 别名(宏)

在引擎根目录下的macro.sl中配置宏
你应该会看到很多已经编写好的宏,新建一个空白行以编写你自己的宏
宏在剧本中有非常丰富的应用场景,以下是常用的对话命令的实际表示

@say text={1} name={2} file={3} ===
@preview text={1}
~{2}@name name={2}
~{2}@backlog text={1} name={2} file="./static/AudioClip/{3}.flac"
~!{2}@backlog text={1} name={2}
~{1}@audio type="Clip" file="./static/AudioClip/{3}.flac"
#say text={1} name={2}
@icon

宏的展开顺序是从上到下,每个宏之间使用----分割
可以注意到,宏可以为命令参数设置默认值,为命令和参数设置别名,将多个命令合成为一个命令
如果一个命令参数的类型为文本,那么可以用插值语法来简化文本中的重复部分
